## 3 параграф. Свойства вероятностей для полной группы

Опр 3.1
События H1, H2, ..., Hn образуют полную группу непопарно несовместных событий (полную группу), если они попарно несовместны:
Hi, Hj = /0, i != j, i = 1,n
сумм Hi = $\Omega$
P(Hi) > 0, i=1,n, то H1, ..., Hn - системная гипотиза

Опр 3.2
Пусть с опытом G связано полная группа событий таких, что вероятность каждого из них одинаковая. Тогда говорят, что опыт G сводится к схеме случаев
Если при этом случайному событию A благоприятствует m событий Hi, i=1,n, то эти события называются благоприятствующими случаями

Пример:
Gj - подбрасывание игрального кубика
$\Omega1$ = {$\omega$i = {выпало i}}, i=1,n
$\Omega1$ - схема случаев; пространство элементарных событий, система гипотез
$\Omega2$ = {$\omega$i = {выпало i}}, i=1,7
$\Omega2$ - пространство элементарных событий, полная группа, не является схемой гипотез и схемой
$\Omega3$ = {$\omega$i = {выпало i}}

Опр 3.3
Условной вероятностью события A относительно события B называется вероятность наступления события A при условии, что событие B произошло
P(A|B) = $P(AB)/P(B)$
Замечание:
Условных событий не бывает, бывают только условные вероятности. Это означает, что  $P(A+B|CD) = P((A+B)CD) / P(CD)$

## Параграф 4. Основные формулы для вычисления вероятностей

### 4.1 Теорема умножения вероятностей
Вероятность произведений события A1, ..., An равно произведению условных вероятностей
P(ПAi) = P(A1)P(A2|A1)P(A3|A1A2)... P(An|A1A2...An-1)
Доказательство:
По индукции
ММИ - 1, n = 2
P(A1A2) = P(A1)P(A2|A1) по св-ву 17

ММИ - 2, n = k
Предположим, что ( * ) верна для n = k, т.е. P(ПAi) = P(Ai)P(A2|A1)...P(An|A1A2...)

ММИ-3, n = k + 1
 Предположим, что ( * ) верная для n = k + 1 с учетом ММИ-2
P(ПAi) = P(A1...AkAk+1) = P(A' Ak+1) по ММИ-1  = P(A')P(A1...Ak|A') с обратной заменой = P(A1...Ak)P(Ak+1|A1...Ak) по ММИ-2 = P(A1), ...,  P(Ak| A1...Ak-1)P(Ak+1|A1...Ak) - ч.т.д.
A' = A1...Ak

Опр 4.1
Событие A и B называются независимыми, если вероятность их произведений равна произведению их вероятностей

Опр 4.1.2
События A1,..., An называются попарно независимыми если любые два из них независимы

События A1, ... An называются независимыми если для любых k = 2,n выполняется P(A1, ...A_i_k) = P(Ai) ... P(A_i_k), i=1,n

Замечание:
Из независимостей событий следует их попарная независимость.
Если A1 ... An - н/з => попарно н/з
Обратное неверно!

Пример:
Тетраэдер
гр.1 - синий
гр.2 - красный
гр.3 - краный
гр.4 - все три цветы
Пусть опыт G состоит в подбрасывании тетраэдера и наблюдении цвета грани, на которую он упал
А = {грань содержит красный цвет}
B = {грань содержит синиц цвет}
C = {грань содержит зеленый цвет}
Проверим, являются ли A, B, C попарно независимыми и попарно н/з
P(A) = 1/2
P(B) = P(C) = 1/2
P(AB) = 1/4 
P(A)P(B) = 1/4 из <- и P(AB)  следует, что A,B - н/з
Аналогично A, C и B, C - н/з => A, B, C - попарно н/з

P(ABC) = 1/4
P(A)P(B)P(C) = 1/8 <- и P(ABC) => не явл п/з в совокупности

Замечание:
Если, событие A и B такие, что их вероятности отличны от нуля - не совместны, то А и В - завис
P(AB) = P(A)P(B)
P(AB) = 0 т.к. A, B - не совместны => P(AB)!= P(A)P(B) т.е AB - зависимы
п/с -> завис
с -> завис / не завис

Пример:
Вытаскиваем карту из колоды

A, B - совместны
P(A) = 1/2, P(A|B) = 1/2, P(A|B) = 1/2, P(A|не(B)) = 1/2
A, B - п/з

### 4.2 Теорема сложения

Вероятностно суммы событий А и B определяется формулой сложений вероятностей
P(сумм Ai) = сумм P(Ai) - сумм_i<j P(AiAj) + сумм_i<j<k P(AiAjAk) -...+(-1)^(n-1) P(A1...An)
Докозательство дз
Пример:
P(A1+A2+A3) = P(A1) + P(A2) + P(A3) - P(A1A2) - P(A2A3) - P(A1A3) + P(A1A2A3)
Заметим, что P(суммAi):
	Если Ai - н/с, то == сумм P(Ai)
	Если Ai - совместны, то == 1 - P(не(суммAi)) = 1 - P(ПAi)
		Ai - н/с  то == 1 - ПP(не(Ai))
		Аi - з то == 1 - P(не(A1))P(не(A2)|не(A1))...
