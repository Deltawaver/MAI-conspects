Данный репозиторий - хранилище моих конспектов в обсидиане. Для просмотра крайне рекомендуется использовать Obsidian, предварительно клонировав этот репозиторий к себе на устройство командой ниже:
```
git clone https://github.com/Deltawaver/MAI-conspects
```

Конспектирование лекции проходит через 3  этапа:
1) Сплошная запись на лекции. Времени на форматирование не хватает, поэтому все имеет "сырой" вид со сплошным текстом (по крайней мере я пока что не наловчился)
2) Восполнение упущенных частей конспекта. Если я что-то не успел записать по какой-то причине и при этом у меня есть ещё чей-нибудь материал с лекции, то я дополняю + перепроверяю свои записи
3) Обработка конспекта с использованием нейросетей. ИИ приводит все в читабельный вид, помогает структурировать все это, а также добавить какие-то пояснения в случае чего. Поскольку нейросети далеко не идеальны, я сам также перепроверяю их вывод на предмет корректности

На текущий момент имеются следующие предметы:
- Системное программирование (up-to-date, ждет дополнения и обработки)
- Социология (Более не ведется)
- Теория вероятности и математическая статистика (up-to-date)
- Фундаментальные алгоритмы (отставание на 1 лабу)
