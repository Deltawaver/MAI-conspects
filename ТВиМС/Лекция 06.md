## Характеристическая функция
Опр 1.1
Характеристическая функция СВ -  это комплексно значная функция вида $g_\xi (t) = M[e^(it\xi)], t принадлежит R, i^2 = -1$ . $g_\xi$ также как и функция с определением является универсальной формулой закона распределения (т.е. существует всегда для любых СВ)
### Свойства Характеристической функции
1) $|g_\xi(t)| \leq 1$
2) $g_\xi(0) = 1$
3) $\nu^r_\xi = 1/i^r * d^r / dt^2 * g_\xi (t) |_t = 0$
4) $\nu = A\xi + B$, $g_\xi(t) = M[e^(it\nu) = M[e^it(A\xi+B) * e^i(At)\xi] = e^itB g|\xi (At$

## Параграф V. Квантиль

Опр 1.2
Квантилиум уровня p малое распредедленяи случайно величины кси называется минимальное число $x_p$ такое, что $x_p = тип {x: F_\xi(x) \geq p} = тип {x: P(\xi \leq) \geq p}$
Замечание:
На интуитивном уровне квантиль понимается как точка распределения всего множества реализаций на две части

| $\xi$ | -1  | 0   | 1   | 2   |
| ----- | --- | --- | --- | --- |
| p     | 1/5 | 2/5 | 1/5 | 1/5 |
$F_\xi (x) = {0, x <-1; 1/5, -1 \leq x < 0; 3/5, 0 \leq x < 1; 4/5, 4 \leq x < 2; 1, x \geq 2 }$
$x_(1/2)$:
$F_\xi (x)= (0) < 1/2$ при x < -1
$F_\xi(-1) = 1/5 < 1/2$
$F_\xi(-1) = 1/5 < 1/2$ при -1 <= x < 0
$F_\xi(0) = 3/5 > 1/2$ => x_(1/2) = 0

## Параграф 6. Основные дискретные распределения
### 6.1 Биномиальное распределение
Дискретная случайная велечина кси распрделена по биномиальным законам с характеристиками n, p
$\xi ~ Bi(n; p)$
$P(\xi = k) = C^k_n p^k (1-p)^(n-k)$
$k = 0, 1 ..., n$
С практической точки зрения случайная величина, распрделенная по по биноминальному закону ($\xi ~ Bi(n; p)$) характеризует число появлений события A (успех) серии из n независимых испытаний
#### Свойства $\xi ~ Bi(n; p)$ :
1) Условия нормирования

| \xi | 0   | 1... | ... | n   |
| --- | --- | ---- | --- | --- |
| p   | p_0 | p_1  | ... | p_n |
$p_i = c^i_n p^i (1 - p)^(n - i)$
$\sum^n_(i=0)p_i = \sum^n_(i=0)C^i_n p^i (1-p)^(n-i) = (p+(1-p))^n = 1$
2) Функция распределения
	$F_\xi (x) = \sum^n_(k=0) l(x-k) C^k_np^k(1-p)^(n-k)$
